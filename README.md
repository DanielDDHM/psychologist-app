## 💻  Getting Started 

1. Clone the Repo;
2. Install development dependencies: `$ npm install`
3. Check the env vars (in the envs folder).
4. Create a .env file in the root of the project.
5. Start development server: `$ npm run dev`
6. Enjoy!
